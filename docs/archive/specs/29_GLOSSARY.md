# Glossary

| Term | Meaning |
|---|---|
| Adapter | Stateless connector that fetches data from a source and emits a normalized envelope. |
| Agent | AI-assisted module that recommends or validates decisions without bypassing policy. |
| Benchmark | Labeled evaluation set used to measure quality, latency, cost, or accuracy. |
| Canonical URL | Normalized preferred URL for a page after removing noise and resolving variants. |
| Crawl Policy | Deterministic service that decides whether a URL may be crawled. |
| Deduplication | Process of identifying duplicate or near-duplicate URLs, documents, media, or entities. |
| DLQ | Dead Letter Queue for failed events that cannot be processed normally. |
| Entity | Canonical person, company, place, paper, product, domain, or organization. |
| Entity Resolution | Linking a mention to the correct canonical entity. |
| Evidence Graph | Data structure linking claims, sources, documents, entities, and provenance. |
| Extraction Quality | Score indicating whether extracted content is complete and useful. |
| Fast Mode | Low-latency mode using cache/index/entity card only. |
| Freshness Score | Score measuring how recent or current a result/document is. |
| Hybrid Retrieval | Retrieval using lexical, semantic, metadata, and entity-aware ranking. |
| Intelligence Card | Structured summary of an entity/topic with evidence, sources, related entities, and signals. |
| MinIO | S3-compatible object storage used for raw and processed files. |
| OpenSearch | Search engine used for indexed trusted data. |
| Provenance | Traceable record of where data came from and how it was processed. |
| Quality Score | Trust score composed of relevance, reliability, freshness, authority, entity match, and risk. |
| RAG | Retrieval-Augmented Generation, where AI answers using retrieved source documents. |
| Source Reliability | Score describing how trustworthy/useful a source is for a vertical. |
| Trust Decision | Accept, Review, Reject, Index, Crawl, or Quarantine decision based on scores and policies. |
| Vertical Pack | Productized use-case bundle such as Company Intelligence or RAG Dataset Builder. |
